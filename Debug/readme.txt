In order to run this file, you shall follow the next steps:
* Copy "res/" folder here.
* Download "freeglut 3.0.0 MSVC Package" from here https://www.transmissionzero.co.uk/software/freeglut-devel/, uncompress it and copy "freeglut.dll" file inside freeglut/bin/ (not x64) into this folder.
* Double-click the runnable (or .exe).
